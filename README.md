# PlayerSpeed MC[1.16.X]
PlayerSpeed is a minecraft plugin. It permit you to change the creative player fly speed and that's all ! (I know that there is already Essential doing this but i didn't want to use it for conflict with other plugins)

# Commands: 
- /speed [number between 1 & 10]

# Permissions 
- playerspeed.*     [Use all commands plugin]
- playerspeed.speed [Use command /speed]
